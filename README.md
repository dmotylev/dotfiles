# Dmitry's dotfiles

## Installation

### Git-free install

To install these dotfiles without Git:

	curl -Ls http://git.io/... | tar xzs '/dmotylev-dotfiles-[a-f[:digit:]]*/dotfiles/' && dotfiles/bootstrap

or even shorter

	eval $(curl -Ls bit.ly/install_dots)

To update later on, just run that command again.
